### ▶️ Start the app
```bash
npm start

🧪 Run tests
npm test

Launches the test runner in interactive watch mode.

npm run build

Builds the app for production to the build folder.
It optimizes the build for best performance and minifies the output.

Your app is ready to be deployed!

⚠️ Eject (advanced)
npm run eject

Note: This is a one-way operation. Once you eject, you can't go back.

It copies all configuration files and dependencies so you have full control over them.
